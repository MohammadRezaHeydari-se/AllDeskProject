# VoiceForge Development Roadmap

## Version 1.0 (Current) - MVP
- [x] Text-to-Speech generation (up to 5000 chars)
- [x] 10 voice profiles (5 male, 5 female)
- [x] Voice preview
- [x] Audio export to custom location
- [x] Built-in audio player
- [x] Progress bar with ETA
- [x] Chunk-based generation
- [x] GPU detection (CUDA/Metal/CPU)
- [x] Cross-platform dependency setup
- [x] Structured JSON logging
- [x] Log viewer UI
- [x] Dark theme UI
- [x] Configuration persistence

- [ ] Actual XTTS v2 model integration
